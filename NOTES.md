GIGME CLI

1. User is greeted by Gigme New York
2. user gets a list of New York loactions
3. user is asked to input a location
4. user gets a numbered list of gig catagories
5. user selects a catagory by number
6. user gets a list of 10 gigs in that area under that catagory
7. user can select the number of that gig to get more info
8. user can go back to results list, access the next ten gigs (up to 50), go back to gig catagories, or exit
